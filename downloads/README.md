# StockFlow Android APK

Arquivo de instalacao direta para Android:

- `StockFlow-v1.0.0-android-release.apk`
- Pacote: `com.raphaelzarour.stockflowmobile`
- Versao: `1.0.0`
- SHA256: `C0EA2ACC347D0A89506E9312CF38192ACBE2CA1CFB1006E5758545E78DE0EF7D`

## Como instalar

1. Abra o link do APK no celular Android.
2. Baixe o arquivo.
3. Toque no APK baixado.
4. Se o Android bloquear, permita a instalacao por fonte desconhecida para o navegador ou app de arquivos.
5. Conclua a instalacao e abra o StockFlow.

O app usa o Supabase real configurado no build.
